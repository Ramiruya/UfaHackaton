# @UfaSigmabot

Проект телеграм-бота UfaSigmabot от команды Sigma для Ufa_Hack2024!

Бот создан для автомаизации процесса взаимодействия с пользователями сервиса. Его функции - предоставление частых вопросов и ответов, а в случае, если не нашелся ответ на вопрос, то можно обратиться к оператору.

Оператор, в свою очередь, может общаться с пользователем напрямую через бота.

Использована библиотека telebot. Реализована возможность использования базы знаний ЧаВО в формате json.
